## :sparkles: Features

-   Tests with Jest
-   Continuous integration with CircleCI
-   Code coverage with Codacy
-   ECMAScript 2015+ code compiling with Babel
-   Linting with eslint, prettier and Airbnb style guide
-   Static type checking with Flow
